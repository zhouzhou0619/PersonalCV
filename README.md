# Yihan Zhou's Resume Website

# Run Project

### 1. Clone the project

### 2. Run the project

```shell
npm i --save --legacy-peer-deps
npm start
```

### 3. Build

```shell
npm run build
```

### 4. Deploy

```shell
npm run deploy
```
